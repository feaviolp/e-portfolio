Summary Post

It was interesting that van Toor (2023a) and I both took a similar approach by producing a stacked bar chart. It’s a shame that there are not more responses to compare.

Van Toor (2023b)  usefully suggested using geom_col instead of geom_bar because it doesn’t required stat=’identify’ to be specified, which is a useful little time saver, although the outcome would be identical.

A leftover comment in my code was also spotted (van Toor, 2023b) which I had omitted to remove. That was a good observation and proved that Astrid read through the code, so in a way it was positive even though it was my error.

Van Toor (2023c) confirmed that, like me, she had to transform the dataset in order to make it plot the stacked chart correctly. She did, however, have a much more elegant approach than me which I would try in the future.

Similarly, it was interesting that the solution in van Toor (2023a) to sequencing the statements was much more elegant than my option of adding a prefix and alphabetically sequencing them. I would also try this if I were to repeat the exercise.

I preferred my stack sequencing with ‘no opinion’ in the middle though, which van Toor (2023c) agreed was a good idea.

In summary, it was a useful exercise where Astrid and I took a similar approach with slightly different options and I think we both learned a bit from each other so we’d both do it better next time.
